## ThystTV 1.1.5

### Highlights
- Hotfix release for a stream-switching crash that could happen when jumping between live streams quickly.
- Centralized player teardown so only the active player fragment is allowed to close the current session.
- Hardened Media3 and ExoPlayer cleanup paths to avoid unsafe item removal during fast switches.

### Included In This Release
- Moved player shutdown responsibility into the guarded `MainActivity.closePlayer(...)` path.
- Removed redundant direct `close()` calls before fragment replacement and close gestures.
- Replaced indexed media-item removal with safer full-list clearing for player backends that support it.

### What To Watch
- Stream-to-stream switching from raids, chat actions, channel pages, and suggested/live lists.
- Sleep timer expiration, swipe-to-close, and minimized-player close behavior.
- Stream to VOD and VOD to stream transitions.

### Release Type
- Hotfix release focused on playback stability rather than feature changes.
