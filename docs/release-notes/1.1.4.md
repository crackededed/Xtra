## ThystTV 1.1.4

### Highlights
- Reworked the Stats tab into an adaptive dashboard that behaves much better across phone portrait, phone landscape, tablets, and rotation while the screen is already open.
- Merged top channels and loyalty/session information into a single Favorite Channels card with clearer ranking and progress indicators.
- Improved player state handling around brightness restore, floating chat mode, and fullscreen transitions.
- Synced selected upstream Xtra improvements for queries, download quality, strings, launcher assets, and build maintenance without pulling in the unstable upstream player rewrite.

### Included In This Release
- Width-based stats layouts for `600dp` and `840dp` breakpoints, plus removal of the old separate tablet stats screen path.
- Rotation refresh fixes so the dashboard re-lays out immediately when the device orientation changes.
- New tests for stats span/layout behavior and player brightness/chat state helpers.
- Safer stream download quality handling and GraphQL/query updates from upstream Xtra.
- Launcher icon refresh and centering cleanup.

### Upstream Sync Status
- ThystTV includes selected upstream Xtra fixes from the recent upstream cycle, but some of them were cherry-picked and conflict-resolved locally instead of merged as the original upstream commits.
- Because GitHub compares commit ancestry rather than just file contents, the fork banner can still show ThystTV as "behind" even when those safe upstream changes are already included here.
- The main upstream item still being intentionally deferred is the upstream player rewrite, which is being held back until it can be integrated without regressing ThystTV's custom player, gesture, stats, and floating-chat behavior.

### Still To Do
- Final on-device regression testing for seek gestures, playback-speed gestures, and brightness behavior on phone and tablet.
- Split-screen and arbitrary window-resize verification on large screens.
- Final wide-tablet density and span polish after more real-device screenshots.
- Broader upstream player-stack changes are still deferred until they can be merged without breaking ThystTV-specific player behavior.
